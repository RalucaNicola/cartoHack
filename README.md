### Gestaltung einer digitalen Karte: ein kollaboratives Experiment

Digitale Karten sind toll! Aber wie erstellt man eine Web Karte? Das werden wir in diesem CartoHack zusammen anschauen und eine Karte anhand moderner Web Technologien von Grund auf erstellen.
